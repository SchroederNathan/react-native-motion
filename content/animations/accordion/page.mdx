## Overview

The Accordion smoothly expands and collapses content sections using animated height transitions. A chevron icon rotates to indicate the current state, and the height animation uses layout transitions for buttery performance.

## Usage

```jsx
import { Accordion } from 'react-native-motion'

export default function App() {
  return (
    <Accordion
      sections={[
        { title: 'Section 1', content: <Text>Content 1</Text> },
        { title: 'Section 2', content: <Text>Content 2</Text> },
        { title: 'Section 3', content: <Text>Content 3</Text> },
      ]}
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| sections | Section[] | required | Array of `{ title, content }` objects |
| allowMultiple | boolean | false | Allow multiple sections open at once |
| initialOpen | number[] | [] | Indices of initially open sections |
| springConfig | SpringConfig | `{ damping: 15, stiffness: 120 }` | Spring configuration |
