## Overview

Toasts slide in from the top with a spring animation, stack with slight scale and offset, and can be dismissed with a horizontal swipe gesture.

## Usage

```jsx
import { ToastProvider, useToast } from 'react-native-motion'

function App() {
  return (
    <ToastProvider>
      <MyScreen />
    </ToastProvider>
  )
}

function MyScreen() {
  const toast = useToast()

  return (
    <Button
      title="Show Toast"
      onPress={() => toast.show({ message: 'Hello!' })}
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| message | string | required | Toast message text |
| duration | number | 3000 | Auto-dismiss duration in ms |
| position | `'top' \| 'bottom'` | 'top' | Stack position |
| maxVisible | number | 3 | Maximum visible toasts |
