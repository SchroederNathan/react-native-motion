## Overview

The Spring Modal combines a fading backdrop with a spring-animated content panel. The modal scales up from 0.9 with a slight vertical offset, creating a natural "popping in" feel.

## Usage

```jsx
import { SpringModal } from 'react-native-motion'
import { useState } from 'react'

export default function App() {
  const [visible, setVisible] = useState(false)

  return (
    <>
      <Button title="Open" onPress={() => setVisible(true)} />
      <SpringModal
        visible={visible}
        onClose={() => setVisible(false)}
      >
        <Text>Modal content here</Text>
      </SpringModal>
    </>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| visible | boolean | false | Controls modal visibility |
| onClose | `() => void` | required | Called when backdrop is tapped |
| backdropOpacity | number | 0.5 | Maximum backdrop opacity |
| springConfig | SpringConfig | `{ damping: 20, stiffness: 300 }` | Spring configuration |
