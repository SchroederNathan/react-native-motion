## Overview

Swipe to Delete lets users reveal a destructive action by dragging a list row horizontally. The row follows the finger with a damped spring, and snaps open or shut depending on the release velocity and distance.

## Usage

```jsx
import { SwipeToDelete } from 'react-native-motion'

export default function App() {
  return (
    <SwipeToDelete onDelete={() => console.log('Deleted!')}>
      <Text>Swipe me</Text>
    </SwipeToDelete>
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| onDelete | `() => void` | required | Called when the delete action is confirmed |
| threshold | number | 0.4 | Swipe distance ratio to trigger delete |
| springConfig | SpringConfig | `{ damping: 20, stiffness: 200 }` | Spring physics configuration |
| deleteColor | string | '#FF3B30' | Background color of the delete action |
