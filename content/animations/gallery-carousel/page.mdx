## Overview

The Gallery Carousel uses spring physics to create natural-feeling swipe interactions. Images snap into place with configurable tension and friction values.

## Usage

```jsx
import { GalleryCarousel } from 'react-native-motion'

export default function App() {
  return (
    <GalleryCarousel
      images={[
        { uri: 'https://example.com/image1.jpg' },
        { uri: 'https://example.com/image2.jpg' },
        { uri: 'https://example.com/image3.jpg' },
      ]}
      snapInterval={300}
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| images | ImageSource[] | required | Array of image sources |
| snapInterval | number | 300 | Distance between snap points |
| springConfig | SpringConfig | `{ tension: 200, friction: 20 }` | Spring physics configuration |
| onIndexChange | `(index: number) => void` | - | Callback when active index changes |
